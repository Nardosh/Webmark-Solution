# Webmark Solution 🌐

**Webmark Solution** is a modern and responsive website built to showcase professional web development services for businesses, schools, restaurants, and freelancers.

This project is designed as a portfolio and lead generation website where potential clients can explore services and contact the agency.

---

## 🚀 Features

* Modern responsive design
* Hero section with animated illustration
* Service showcase section
* Portfolio projects display
* Pricing section
* Contact form with email notifications
* Smooth scrolling navigation
* Mobile responsive layout
* Floating WhatsApp contact button

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* AOS Animation Library
* FormSubmit (for contact form email notifications)
